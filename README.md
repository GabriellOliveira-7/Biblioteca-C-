# 📚 Sistema de Acervo de Biblioteca

Sistema desenvolvido em C/C++ para gerenciamento de um acervo de livros, utilizando armazenamento em arquivo binário para salvar os dados automaticamente.

O projeto foi desenvolvido com foco em praticar lógica de programação, estruturas de dados, manipulação de arquivos e modularização em C.

---

# 🚀 Funcionalidades

✅ Cadastro de livros  
✅ Listagem completa do acervo  
✅ Pesquisa de livros por código  
✅ Ordenação por ano de publicação  
✅ Salvamento automático em arquivo `.dat`  
✅ Carregamento automático dos dados ao iniciar o sistema
